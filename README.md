# Dynamic-Weather-app
Dynamic Weather App built with JavaScript. Fetches real-time data from a Weather API based on user input or geolocation. Clean, responsive UI.
Features
Real-time Data: Fetches up-to-the-minute weather conditions, temperature, and forecasts.

Location Lookup: Allows users to search for weather by entering a city name.

Geolocation Support: Automatically detects and displays weather for the user's current physical location.

Detailed Information: Displays key metrics including:

Current Temperature (in Celsius/Fahrenheit).

Weather Condition (e.g., Sunny, Cloudy, Rain).

Humidity percentage.

Wind Speed and Direction.

'Feels Like' Temperature.

Dynamic UI: Background and icons dynamically change based on the current weather condition for an engaging user experience.
